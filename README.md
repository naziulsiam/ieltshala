# IELTShala — AI-Powered IELTS Prep for Bangladesh

Bangladesh's first AI-powered IELTS preparation platform. Practice Listening, Reading, Writing & Speaking with instant AI feedback, mock tests, and personalized study plans.

## Tech Stack

- **Vite** — fast build tooling
- **React 18** + **TypeScript**
- **Tailwind CSS** + **shadcn/ui**
- **React Router** — client-side routing
- **Recharts** — data visualization
- **Tanstack React Query** — data fetching

## Getting Started

```bash
# Clone
git clone https://github.com/naziulsiam/ieltshala.git
cd ieltshala

# Install dependencies
npm install

# Start dev server
npm run dev
```

The app runs at **http://localhost:8080** by default.

## Scripts

| Command | Description |
|---------|-------------|
| `npm run dev` | Start dev server |
| `npm run build` | Production build |
| `npm run preview` | Preview production build |
| `npm run lint` | Run ESLint |
| `npm test` | Run tests |

## Deployment

This project is configured for **Vercel**. Simply import the repo and deploy — `vercel.json` handles SPA routing automatically.
